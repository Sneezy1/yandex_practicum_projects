# Учебные проекты Яндекс.Практикум 
В данном репозитории представлены проекты, которые были выполнены в время обучения в Яндекс.Практикум на курсе ["Специалист по Data Science"](https://practicum.yandex.ru/data-scientist/)

Каждый проект содержит описание задания, задачу и подробное описание решения этой задачи.

При выполнении проектов были изучены следующие темы:
* Разработка на языке Python
* Предобработка данных
* Исследовательский анализ данных
* Статистический анализ данных
* Машинное обучение(модели, метрики)
* Применение машинного обучения в задачах бизнеса
* Обучение с учителем
* Линейная алгебра
* Численные методы
* Временные ряды
* Обработка естественного языка

* Компьютерное зрение
